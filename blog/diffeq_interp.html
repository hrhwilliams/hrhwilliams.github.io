An Interpreter for Differential Equations, Part 1

July 17, 2021

Ordinary Differential Equations

To keep things simple, I’m only talking about ordinary differential equations (ODEs), which are differential equations of a single variable. An ordinary function of a single variable expresses a relation between one value and another, $y(x)$. Graphically, this gives you a bunch of points $(x, y(x))$ that trace one curve. On the other hand, an ODE often depends on both $x$ and the current value of the function $y(x)$, and for higher order ODEs, one or more derivatives of $y(x)$. Given the point $(x, y(x))$ for a hypothetical solution $y(x)$, an ODE returns the slope at that point. I say ‘hypothetical’ because an ODE can have not just one function as its solution, but an entire family. For example, the family of solutions to the first-order ODE

\[ \frac{\text{d}y}{\text{d}x} = -2xy \]

can be found by multiplying both sides by $\text{d}x$, dividing by $y$, and integrating:

\[\begin{aligned} &\int \frac{\text{d}y}{y} = -\int 2x\,\text{d}x \\ &\ln y = - x^{2} + c \\ &y = e^{-x^{2} + c} \\ &y = ce^{-x^{2}}, \end{aligned}\]

and setting the constant \(c\)<\span> to any number gives you a function that solves the initial equation, as <\p> \[ \frac{\text{d}}{\text{d}x}\left(ce^{-x^{2}}\right) = -2x\left(ce^{-x^{2}}\right). \]

Finding a particular solution, i.e.<\i> the solution $y(x)$ that goes through the point $(x, y(x))$ involves specifying an initial value of the function at $x$ and solving for the constant. If you think of the ODE as itself a function of two variables, <\p> \[ \frac{\text{d}y}{\text{d}x} = f(x, y(x)), \]

then there’s a value at any point on the plane, just with the catch that the second variable is dependent on the first, so that once you pick a point, you are constrained to the curve given by $y(x)$. <\p>

Euler’s Method

def step(f, t_n, y_n, step_size):
    return y_n + (step_size * f(t_n, y_n))


def eulers_method(f, t_initial, y_initial, t_final):
    step_size = 0.01
    t, y = [t_initial], [y_initial]
    t_n, y_n = t_initial, y_initial

    while t_n <= t_final:
        t_n += step_size
        y_n = step(f, t_n, y_n, step_size)
        t.append(t_n)
        y.append(y_n)
    return t, y
